# craft-lamp

# Install
`$ npm install craft-lamp`

# Parameters
```sh
topResolution - determines how many sides the lamp cover will have.
bottomResolution - determines how many sides the lamp base will have.
```

# Example
```html
<craft>
	<craft name="lamp" module="craft-lamp"/>
	<lamp></lamp>
</craft>
```
